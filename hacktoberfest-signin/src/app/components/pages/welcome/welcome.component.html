Welcome to Hacktoberfest 2021

Every year Digital Ocean hosts an awesome Hackathon for Github users to learn to collaborate and contribute to open source projects.

In the spirit of Hacktoberfest, I have created a first time project for new and old users to assist them in achieving their quota of Pull Requests (PRs) this Hacktoberfest.

This project features Github users and their experience in each language in a visual graph, this will program will ask for your Github username and your programming language experience.

In order to participate in hacktoberfest you will need to make a total of 4 PRs in opensource projects throughout Github.

In order to make a valid pull request in this project follow the documentation inside the README of this repo.

This project can help with your goals this year by adding yourself to the list of users who have added their language preferences.


Happy Hacking!
